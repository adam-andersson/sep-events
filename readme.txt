The project is deployed at: https://adam-andersson.github.io/sep-events/

Requirements: 
* Node v16.17.0 or nvm
* NPM (should be included in Node)

Setup:
* If using nvm, in terminal cd to the root directory of the project and run: "nvm use" to start using Node v16.17.0.
* In terminal, run "npm install" to download the node_modules used in the project.
* In terminal, run "npm start" to launch the development server. It will be available at: http://localhost:3000/sep-events per default.

Testing:
* In order to run our test suite, from the root directory, in the terminal run "npm test" -> press 'a' key to run all tests.

Notes:
* This project was bootstrapped with Create React App. Reference https://github.com/facebook/create-react-app for more information.
